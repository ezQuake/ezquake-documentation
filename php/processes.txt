This text describes how scripts generating and maintaining on-line manual works.
johnnycz, 2005-07-22

Input data for scripts:
1) configs with default values 
      config (made by cfg_save) contains all variables with default
      settings (made by cfg_reset) and variable groups
2) variable descriptions stored in xml files
3) definition of major groups

Output data:
  *.inc files used in HTML manual
