First off, let's assume your Quake directory is C:\quake. Note that there is also eQuake pack out there.

How do I install ezQuake?

  1. Make sure you have pak0.pak in C:\quake\id1
  2. If you have used fuhquake, rename (or copy) C:\quake\fuhquake to C:\quake\ezquake
  3. Unzip the ezQuake package into your Quake root (C:\quake)
  4. Unzip the Security Module into your Quake root too (C:\quake)
  5. If you are using Windows OpenGL version for the first time, you should also download First run package and unzip it into your Quake root too (C:\quake)

This is how it should look like:

  • C:\quake
  • C:\quake\ezquake.exe
  • C:\quake\libjpeg.dll
  • C:\quake\libpng.dll
  • C:\quake\zlib.dll
  • C:\quake\ezquake-gl.exe
  • C:\quake\ezstart.exe
  • C:\quake\id1\pak0.pak
  • C:\quake\ezquake\pak0.pak

There might be more files than this, but these are the required files

I get a stupid gfx.wad error message! wtf?!

There are several things that can cause this:

  1. You don't have C:\quake\id1\pak0.pak
  2. You have opened C:\quake\id1\pak0.pak in some program, i.e. Pak Explorer (hence you need to close that program)
  3. You have unzipped the ezquake package into the wrong directory (ezquake.exe and ezquake-gl.exe should be in C:\quake)
  4. You have created a shortcut with the wrong "Start in" directory
