Features
  • Changelog / Feature list - what's new and cool in this client
  • Head Up Display - how to customize your in-game HUD, move, resize, group, show and hide screen elements.
  • Graphics - graphical settings based on FuhQuake's manual + full description of new AMFQuake + TEI commands/variables.
  • Multiview - watch Multi View Demos (MVD) from up to four point of views.
  • Independent physics - render the game with as much frames per second (FPS) as your hardware allows.
  • Server Browser - improved MQWCL's Server Browser, most missing feature in FuhQuake.
  • MD3 Models - download some from QuakeWorld.ru server.
  • Video stream capture - instantly capture xvid/divx videos with mp3 sound
  • Scripting - basic & advanced scripting possibilities.
  • Linux: Real Time Clock, EVDEV support
  • Keymap - this feature allows you to use non-US keyboard layout
  • Features inherited from FuhQuake
    • Configuration Manager - How to save and restore your settings.
    • Spectator Tracking - Description of commands that make spectator tracking more convenient.
    • Logitech Mice - How to enable extra buttons on your logitech mice.
    • Particle Effects - How to enable and customise different particle effects.
    • MP3 Player - How to control the winamp or xmms mp3 players from within the client.
    • Crosshairs - Everything you need to know about customising crosshairs.
    • Match Tools - How to enable automatic demo recording, console logging and screenshot taking.
    • TP Pointing - How to communicate what you are pointing at to your teammates.
    • Linux Information - Information specific to Linux binaries.
    • QC Extensions - Information about QC extensions available in the inbuilt server.
